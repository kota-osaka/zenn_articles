## Puntoneとは

ブラウザだけで動く演奏アプリです。

テルミンのように無段階で連続的に音程が変化します。さらに、和音の表現も可能です。それでいて、ドレミファソラシドは鳴らしやすくしていますので、テルミンよりも演奏しやすいものになっています。

👉 [アプリを開く](https://puntone-synth.pages.dev/)

発音モードは3つあります。初めて開いたときは「コードグライド」で始まります。

| モード | ざっくり言うと |
|---|---|
| **コードグライド**（おすすめ） | 2枚のパッドをなぞって、最大4音のコードをグライドさせながら演奏 |
| **デュアルパッド** | コードグライドの兄弟。音程が半音単位でスパッと切り替わる（切り替わり速度は調整可能） |
| **鍵盤パッド** | 鍵盤＋自由な音程のパッド。テルミンの感覚にいちばん近い |

## 1. 画面の見かた

画面右上に共通のボタンが並んでいます。

- **●**（録音）: 押すと録音開始、もう一度押すと停止
- **一覧**: 録音テイクの一覧パネルを開く
- **目のアイコン**: 録音したテイクや取り込んだ音源の再生中に、音に合わせて背景が光る演出のON/OFF
- **全画面**: 全画面表示にして横向きに固定します（Androidのみ。iPhoneは非対応）
- **⚙**（設定）: 設定パネルを開く。発音モードの切り替えもここ

発音モードの切り替えは **⚙ 設定 → 上部のモードボタン** です。設定値はすべて自動保存され、次回開いたときも引き継がれます。

![設定パネル](/images/puntone-operation-guide/settings.png)

:::message
スマホでは**横向き**での演奏を推奨しています。パッド系のモードを縦持ちで開くと回転の案内が表示されます。
:::

## 2. コードグライド（おすすめ）

左右2枚のパッドをなぞって演奏する、Puntoneの看板モードです。

![コードグライド](/images/puntone-operation-guide/chord-glide.png)

### 左パッド — ルート音と5度

縦のバーが半音ごとの音階です（明るいバー＝白鍵、暗いバー＝黒鍵）。バーの上を押さえている間は、ルート音がその音にぴったり固定されたまま、コードのほうが微分音的に変化します。バーとバーの**すき間**に入ると、ルート音そのものが隣の音へ連続的に変化します。

![なぞって演奏する（グライド）](/images/puntone-operation-guide/glide-trace.png)

縦方向は5度の種類です。上から **-5（♭5）→ 5（完全5度）→ aug（増5度）** が並び、こちらも段の間は連続的に変化します。

### 右パッド — 3度と7度

縦横のバーが交差した市松模様です。縦方向が3度系（**sus4 / Major / minor / sus2**）、横方向が7度系（**Six / 7th / Δ7th / Oct**）。こちらもバーの中心でぴったり、間は連続変化です。

### 鳴る音の数

- 左パッドだけ: **2音**（ルート＋5度系）
- 左右同時（両手）: **最大4音**（ルート＋5度系＋3度系＋7度系）
- 右パッドだけではルートが決まらないため鳴りません

### 音域の移動（パン・ズーム）

パッド上端の、音名ラベルが並んでいる段は「つかんで動かす」ための場所です。ここを横にドラッグすると、音を出さずに音域だけを左右にスライドできます。音域は下は **A0**（ピアノの最低音）まであります。同じ段に2本指を乗せてピンチするとズームします（設定パネルの「ズーム」スライダーでも調整可能）。

![音域を動かす（パン・ズーム）](/images/puntone-operation-guide/pad-pan.png)

なお左パッドでは、発音中にパンを始めたり、パン中に発音を始めたりはできません（誤操作防止）。

### レイアウト調整

パッドの高さは下端のハンドル、左右パッドの幅の比率はパッド間のハンドルをドラッグして調整できます。

## 3. デュアルパッド

コードグライドの兄弟モードです。パッドの構成・パン・ズーム・レイアウト調整はコードグライドと共通で、違いは次の2点です。

![デュアルパッド](/images/puntone-operation-guide/dual-pad.png)

- 左パッドのバーがすき間なく敷き詰められていて、ルート音は**半音単位で**切り替わります（すき間のグライド＝微分音はありません）
- 半音単位ではありますが、切り替わりにかける時間は設定の**ポルタメント**（0〜1000ms）で選べます。0なら即座にスパッと、大きくするほど次の音へなめらかにつながります

音程を正確に取りたいときはこちら、音のつながりを滑らかに歌わせたいときはコードグライド、という使い分けです。

## 4. 鍵盤パッド

鍵盤と自由音程パッドの組み合わせ。テルミンの「空間で音程を取る」感覚にいちばん近いモードです。

![鍵盤パッド](/images/puntone-operation-guide/keyboard-pad.png)

### 鍵盤（上段）

- 音域は **A4〜E6** の20鍵、単音です
- **マウス／タッチ／PCキーボード**のどれでも弾けます（各鍵に対応するPCキーが表示されています）
- 押している間だけでなく、**指を離しても音は鳴り続けます**。止めるには **Stopボタン** か **スペースキー**
- 次の音を弾くと、今の音から滑らかに音程が変化します（ポルタメント）

### ピッチパッド（下段）

- 横方向が音程（鍵盤と同じ範囲を**無段階**で）、縦方向が音量（上ほど大きい）です
- マウスはパッドにカーソルが乗っている間、タッチは触れている間だけ鳴ります
- 鍵盤とは独立した音なので、鍵盤の音に**重ねて**鳴らせます

### 協和度グラフ

鍵盤を押している間、画面上部に鍵盤の音を基準とした協和度のグラフが表示され、パッドで鳴らしている音との音程差が白い点で示されます。点が山の上に来る位置がよく協和する音程です。ハモリの練習にどうぞ。

## 5. 録音・重ね録り・共有

録音は上部バーの録音ボタン ![録音](/images/puntone-operation-guide/btn-rec.png =26x) で開始し、もう一度押すと停止します（最大5分で自動停止）。録音したテイクは、一覧ボタン ![一覧](/images/puntone-operation-guide/btn-takes.png =26x) で開くパネルに並びます。

![録音テイク一覧](/images/puntone-operation-guide/takes-panel.png)

テイクごとのボタンは次のとおりです。オレンジの点1つは「重ね録り」、2つは「重ね録り＋演奏単体テイクも保存」の意味です。

| ボタン | 動作 |
|---|---|
| ![再生](/images/puntone-operation-guide/btn-play.png =36x) | 再生 |
| ![重ね録り](/images/puntone-operation-guide/btn-overdub.png =36x) | **重ね録り**: テイクを再生しながら、重ねて録音（オーバーダブ） |
| ![重ね録り+演奏単体](/images/puntone-operation-guide/btn-overdub-clean.png =36x) | 重ね録り＋**演奏単体のテイクも保存**（ミックスと「(puntone)」ラベル付きの2件が残る） |
| ![ループ再生](/images/puntone-operation-guide/btn-loop.png =36x) | ループ再生（繰り返し再生しながら練習・演奏） |
| ![ループ+重ね録り](/images/puntone-operation-guide/btn-loop-overdub.png =36x) | ループ再生しながら重ね録り |
| ![ループ+重ね録り+演奏単体](/images/puntone-operation-guide/btn-loop-overdub-clean.png =36x) | ループ再生しながら重ね録り＋演奏単体のテイクも保存 |
| ![保存](/images/puntone-operation-guide/btn-download.png =36x) | ファイルとして保存（WebM形式） |
| 共有 | OSの共有シート経由でLINEやメールへ（スマホなど対応環境でのみ表示。PCブラウザでは保存ボタンで代替） |
| ![削除](/images/puntone-operation-guide/btn-delete.png =36x) | テイクを削除 |

パネル右上の ![読み込み](/images/puntone-operation-guide/btn-import.png =90x) からは、手持ちの音声ファイルを取り込めます。取り込んだ音源もテイクと同じように再生・重ね録りに使えます。

- 各テイクには種別ラベル（録音／重ね録り／ループ録り／読み込みトラック など）が付き、日時が同じでも見分けられます
- 保存は最大10件（超過分は古い順に自動削除）。このブラウザだけに保存され、他の端末・他のブラウザとは同期しません
- アプリを閉じても消えず、次に開いたときも一覧に残ります

## 6. 設定

⚙ 設定パネルの項目です。モードによって表示される項目が変わります。

| 項目 | 内容 | 表示されるモード |
|---|---|---|
| 音色 | サイン波／ずんだもん | すべて |
| ズーム | パッドの拡大率（1.5〜16倍）。ピンチ操作とも連動 | コードグライド・デュアルパッド |
| ポルタメント | 次の音へ移り変わる時間（0〜1000ms）。0で即切り替え | デュアルパッド・鍵盤パッド |
| ディケイ | 鍵を弾いたあと、音量が-20dBまで減衰する時間（0.1〜30秒） | 鍵盤パッド |

:::message
コードグライドにポルタメント設定がないのは、バーのすき間によるグライドが常に効いているためです。ディケイがパッド系モードにないのは、パッド系は触れている間鳴り続ける方式のためです。
:::
